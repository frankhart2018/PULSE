# Syntax

## Operators

### Arithmetic operators

1) Addition operator (+):-

```console
1 + 2
```
<b>Result = 3</b>

2) Subtraction operator (-):-

```console
1 - 2
```
<b>Result = -1</b>

3) Multiplication operator (*):-

```console
1 * 2
```
<b>Result = 2</b>

4) Division operator (/):-

```console
1 / 2
```
<b>Result = 0.5</b>

5) Modulo operator (%):-

```console
1 % 2
```
<b>Result = 1</b>

6) Power operator (^):-

```console
2 ^ 3
```
<b>Result = 8</b>
