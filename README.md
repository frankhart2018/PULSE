# Syntax

## Operators

### Arithmetic operators

1) Addition operator (+):-

```console
1 + 2
```
Result = 3

2) Subtraction operator (-):-

```console
1 - 2
```
Result = -1

3) Multiplication operator (*):-

```console
1 * 2
```
Result = 2

4) Division operator (/):-

```console
1 / 2
```
Result = 0.5

5) Modulo operator (%):-

```console
1 % 2
```
Result = 1

6) Power operator (^):-

```console
2 ^ 3
```
Result = 8
